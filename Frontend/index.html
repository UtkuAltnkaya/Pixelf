Get Started

How to pixel an image?

  • First upload image you want to convert.
  • Secondly choose your style: pixel, cubic or linear which will affect the shapes of pixel.
  • After that, choose your blocksize which should be between 1-32. It will affect size of every pixel in image.
  • If you want, you can click grayscale which uses only shades of gray.
  • Similar colors lets image to make closer colors to be same. When you clicked it you have to specify your color number which is between 1-32. This feature will make process slower than normal.
  • Lastly, you can add specific color palette from options. It will uses these colors only on your image. If you not add an color palette it is default 256 different color.
